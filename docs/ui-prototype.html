第 一 章

雨夜归人

雨是子时下起来的。沈砚把刀横在膝上，听檐角的水一滴一滴砸进青石的凹处，像是谁在数着什么。

客栈的灯只剩一盏，照着空荡的堂屋。他要了一壶最劣的酒，却迟迟没有喝——他在等一个人，一个他以为这辈子都不会再等的人。

门被风推开半扇，雨声忽然大了。一个披着蓑衣的身影立在门口，水顺着斗笠的边沿往下淌，看不清脸。可沈砚的手，已经先一步按在了刀柄上。

"十年了。"那人开口，声音比雨还冷，"你的刀，还是这么急。"
